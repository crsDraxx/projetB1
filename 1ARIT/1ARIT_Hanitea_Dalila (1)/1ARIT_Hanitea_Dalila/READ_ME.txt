Explication du code pour le premier algorithme:
    - 1ere etape , on a initialise les valeurs de keyLeft et keyRight dans une classe
    - 2nd etape, on a creer les def cipher(=chiffrement) et uncipher(=dechiffrement) avec le code qui va avec selon le projet
    - 3eme etape, on a creer les instances pour l'affichage des messages chiffrés et dechiffrés
        ps : 
            Les tests comme test_cipher et test_uncipher proviennent de la description et n'étaient pas obligatoires, 
            tandis que cipher_twin, cipherTwin correspondent aux parties demandées dans le projet.


Explication du code pour le second algorithme:
    - 1ere etape , on a initialise trois attributs privés dans une classe (n, offset et direction)
    - 2nd etape, on a creer les def cipher(=chiffrement), uncipher(=dechiffrement) et display_waves(=affichage des vagues) avec le code qui va avec selon le projet
    - 3eme etape, on a creer les instances pour l'affichage des messages chiffrés et dechiffrés
        ps : 
            Les tests comme test_uncipher_wave et test_cipher_wave proviennent de la description et n'étaient pas obligatoires, 
            tandis que cipher_wave correspondent aux parties demandées dans le projet.
